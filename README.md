
# Analyse Statistique Intégrée du Réseau de Bornes Électriques

## Description du Projet

Ce projet présente une analyse statistique complète du réseau de bornes de recharge rapide opéré par ÉnerData Canada dans les provinces du Québec, de l'Ontario et de la Colombie-Britannique. L'objectif principal est d'analyser les différences entre provinces, d'étudier la dynamique de la demande, et d'évaluer l'efficacité de modèles prédictifs pour la gestion du réseau.

## Objectifs de Recherche

1. **Analyse comparative inter-provinciale** : Les différences entre provinces/types sont-elles significatives ?
2. **Dynamique temporelle** : La demande suit-elle une dynamique stable ?
3. **Modèles prédictifs** : Les modèles Markov/ARIMA prédisent-ils efficacement la demande ?
4. **Fiabilité des alertes** : Une alerte est-elle vraiment fiable ?
5. **Visualisations** : Quelles visualisations sont les plus utiles ?

## Technologies Utilisées

- **Python** : Analyse de données et modélisation statistique
- **Pandas** : Manipulation et traitement des données
- **NumPy** : Calculs numériques
- **Matplotlib/Seaborn** : Visualisations
- **Scikit-learn** : Modèles de machine learning
- **Statsmodels** : Modèles ARIMA et analyses statistiques
- **Jupyter Notebook** : Environnement de développement

## Méthodologie

- Analyse exploratoire des données (EDA)
- Tests statistiques pour comparer les provinces
- Modélisation temporelle (ARIMA)
- Modèles de chaînes de Markov
- Évaluation de la fiabilité des systèmes d'alerte
- Création de visualisations interactives
